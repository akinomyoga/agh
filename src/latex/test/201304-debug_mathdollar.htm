LaTeX における全角空白の扱い

例えば \あい などという記述がある場合、 LaTeX は "\あい" をひとまとまりのコマンドと解釈するようである。 然し \　い という記述がある場合には、 LaTeX は "\　" というコマンドに続けて、通常の文字として "い" が来たと解釈する。 つまり、全角空白は全角の他の文字と明らかに異なる扱いを受けている。 取り敢えず、通常の空白文字や記号文字と同じ扱いをすると見て良いだろう。
