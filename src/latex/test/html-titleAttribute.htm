tex:error[title] を使えるか?

tex:error 要素に title 属性を設定し、それを JavaScript で読み取れるかをテストする。

tex:error[title], this.title: [click here!]

  • Cr: OK
  • IE6: OK
  • Fx: OK
  • Sf5.1.7: OK
  • Op12: NG (そもそも xml 要素の onclick が使えない?)

span[title]&tex:error, this.title: [click here!]

  • Op12: OK

span>tex:error[title], this.firstChild.title: [click here!]

  • Op12: OK
