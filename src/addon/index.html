AgehaTeX Extension (agh.addon.aghtex)

概要 ～ メールに書いた TeX を数式で表示 ～

agh.addon.aghtex を使えば、 メールの本文中にある TeX の記述を綺麗な数式で表示する事ができます! agh.addon.aghtex は色々の環境に対する拡張・アドオンとして提供しています。 数式を TeX の文でテキストメール中に書く人は、 これを導入するだけで表示が綺麗になります。と、思います。。

使い方

特定の記号で囲んだ部分を、対応する `$\LaTeX$ の数式・レイアウトに変換して表示します。

始まり間終わり説明
$数式モードの `$\TeX$ (二重改行は不可)$インラインの数式に変換します。
$$数式モードの `$\TeX$ (二重改行は不可)$$独立行の数式に変換します。
\begin{eqnarray*}数式モードの `$\TeX$\end{eqnarray*}複数行の数式に変換します。
`#tex-para#文章モードの `$\TeX$##'`$\TeX$ で整形された文章に変換します。

※ コマンドには一つ一つ対応していますので、対応していないコマンドも沢山あります。 また、\usepackage しないと使えないコマンドは、基本的に未だ対応が進んでいません。

使いたいけれど未対応なコマンドが在れば教えて下さい。 (何かの \usepackage に入っている物でも構いません。使い方や表示の規則なども説明してくれると助かります。) 必ず対応する訳ではありませんが、要望の多い物から順に実装していくつもりです。

長所と短所

基本的にテキストメールに、特定の記号で囲んだ TeX を書いただけの物を対象としています。 メールを閲覧する時に $ で囲まれた部分などを見易く整形して表示するだけで、他は唯のテキストメールと同じです。 この様な形態のために色々の長所と短所があります。

  • ○ 前の人の書いた数式に手を加えて返信出来る。議論に最適
  • ○ メールのサイズが大きくならない

    余分な画像などを添付したりサーバー上にアップロードしたりしなくて済む

  • ○ テキストしか扱えないメーラでも TeX ソースとして扱える。
    • テキストでしか読めない環境では、TeX ソースとして読む事ができる。
    • 対応していないメーラからでも、TeX ソースを埋め込んだメールを送れる。
  • × メールを出す側も見る側も agh.addon.aghtex を入れていないと恩恵を受けられない。
    • 調子に乗って複雑な数式を TeX で書くと、対応していないメーラで見ている人に不親切
    • その為にあらゆるメール環境に agh.addon.aghtex を対応させたいが限度がある。

競合する物。メールで数式を扱う為の物は他にもあります。

  • LaTeX It! :: Add-ons for Thunderbird (Thunderbird 用アドオン)

    これはローカルの TeX 環境を用いて式をコンパイルし、それを画像としてメールに埋め込んで送信する物です。

agh.addon.aghtex4chrome

Google Chrome で GMail を見る時の為の拡張です。

  • GMail, Google Groups, Google Sites で動作
  • 編集時プレビュー機能付き

入れ方

  • 1 以下をダウンロードして保存します。

    [agh.addon.aghtex4chrome.crx]

  • 2 拡張機能タブを開きます。

    アドレスに chrome://extensions を指定するか、 [右上のメニューボタン]-[ツール]-[拡張機能] を選ぶ事で開けます。

  • 3 拡張機能ページの中に 1 でダウンロードした agh.addon.aghtex4chrome.crx をドラッグ&ドロップします。

    何か訊いてきたら内容を確認して OK を押して下さい。

  • 4 自動更新機能を使いたければ拡張機能ページの agh.addon.aghtex4chrome の中の、 [ファイルの URL へのアクセスを許可する] にチェックを入れて下さい。

消し方

  • 1 拡張機能タブを開きます。

    アドレスに chrome://extensions を指定するか、 [右上のメニューボタン]-[ツール]-[拡張機能] を選ぶ事で開けます。

  • 2 拡張機能ページの agh.addon.aghtex4chrome の項目の右に表示されているごみ箱マークをクリックします。

    何か訊いてきたら内容を確認して OK を押して下さい。

agh.addon.aghtex4thunderbird

Thunderbird でメールを見る時の為のアドオンです。

  • テキストメールで動作。HTML メールでは動作しない事にしています

入れ方

  • [agh.addon.aghtex4thunderbird.xpi]
    1. 上からダウンロードして保存する
    2. Thunderbird で [アドオンマネージャ] を開く。([メニュー]-[ツール]-[アドオン] もしくは Alt-T Alt-A)
    3. [アドオンマネージャ] の [拡張機能] を選択
    4. ダウンロードしたファイルを、"アドオンマネージャ" にドラッグ&ドロップ

      ※ 何故か二回ドラッグ&ドロップしないといけない事がある

    5. 確認のダイアログが出るので [インストール] を押す。
    6. (インストールしたらダウンロードした .xpi ファイルは削除して問題ありません。)

何か変な事が起こったら、アドオンマネージャから無効化または削除して下さい。

agh.addon.aghtex4ie

Microsoft Internet Explorer 用のアドオンです。

インストール (Windows)

  1. これをダウンロードして下さい: [agh.addon.aghtex4ie.zip]
  2. agh.addon.aghtex4ie.zip のプロパティを見て [ブロックの解除] ボタンが表示されている場合は、 そのボタンを押してブロックの解除をして下さい。
  3. 適当な所に展開して中の install.bat をダブルクリックして下さい。

    ※ インストール先 (%PROGRAMFILES%\Myoga\agh.addon.aghtex4ie) のディレクトリが開かれます。

  4. ※IE を再起動してから有効になります。
  5. ※ダウンロードした .zip と展開したファイルはもう必要ないので削除して構いません。

アンインストール (Windows)

  1. %PROGRAMFILES%\Myoga\agh.addon.aghtex4ie\uninstall.bat を実行して下さい。

    ディレクトリ %PROGRAMFILES%\Myoga\agh.addon.aghtex4ie を開いて中の uninstall.bat をダブルクリックするか、 ファイル名を指定して実行 ([Win+R]) で %PROGRAMFILES%\Myoga\agh.addon.aghtex4ie\uninstall.bat を指定すれば実行できます。

    %PROGRAMFILES%\Myoga\agh.addon.aghtex4ie 以下に個人のデータを置いている場合には予め退避しておいて下さい。 ディレクトリ毎削除されます。

  2. %PROGRAMFILES%\Myoga ディレクトリが残りますが、中に他に何もない場合には削除して問題ありません。

agh.addon.aghtex4seahorse

対応中

Sleipnir 2 のプラグイン SeaHorse 用の Sleipnir スクリプトです。

インストール (Windows)

  1. SeaHorse を Sleipnir2 に入れていない人は入れて下さい。SeaHorse 入れ方:
    1. ここへ Sleipnir2 で行く: Sleipnir - プラグイン Web ページへの機能追加やデザイン変更などのカスタマイズを可能にする SeaHorse | フェンリル
    2. [ダウンロード] を押す。
    3. 入れますかというダイアログが出るので [OK] を押す。
    4. 再起動しますかと聞かれるので [再起動] で再起動する。
  2. 次に agh.addon.aghtex4seahorse 本体を入れます:
    1. これをダウンロードする: [agh.addon.aghtex4seahorse.zip]
    2. 適当な場所に解凍する。
    3. 中の agh.addon.aghtex4seahorse.user.js と agh.addon.aghtex4seahorse ディレクトリを、 (Sleipnir インストール場所)\plugins\seahorse\ に移動する。

      (Sleipnir インストール場所) は大体 C:\Program Files\Fenrir\Sleipnir 的な所にある。

    4. Sleipnir を再起動する。
    5. (ダウンロードした .zip と解凍した後の残骸はもう不要です。捨てても構いません。)
